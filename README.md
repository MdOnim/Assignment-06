
# 🚀 SwiftDigi-Pro

### 📌 Project Overview

**SwiftDigi-Pro** is a modern React-based web application that empowers users to explore, select, and manage premium digital resources such as AI tools, templates, and productivity software. The platform is designed with a clean interface, responsive layouts, and a seamless user experience.

🔗 **[Live Demo (Netlify)]()**

-----

### 🛠️ Technologies Used

  * **Core:** React.js
  * **Styling:** Tailwind CSS & DaisyUI
  * **Icons:** React Icons
  * **Data Handling:** Axios
  * **Notifications:** React Toastify

-----

### 🚀 Key Features

  * **Smart Cart System:** Effortlessly add products to the cart. Duplicate items are automatically prevented with instant notifications.
  * **Dynamic Product Rendering:** Product data is dynamically fetched from a JSON file using Axios, ensuring scalability and flexibility.
  * **Interactive Notifications:** Real-time success and error alerts powered by React Toastify (e.g., item added successfully or already exists).

-----

### 🎨 UI Highlights

  * **Gradient-driven modern design**
  * **Fully responsive across mobile and desktop**
  * **Smooth hover animations and interactive effects**
